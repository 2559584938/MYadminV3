MYadminV3
===============

> 运行环境要求PHP7.4+，兼容PHP8.1



## 安装

~~~
git clone https://github.com/2559584938/MYadminV3.git
~~~
或者

~~~
git clone https://github.com/2559584938/MYadminV3.git
~~~

运行命令
~~~
composer install
~~~
配置数据库信息

~~~
config/database.php
~~~

导入数据库

~~~
根目录/sql.sql
~~~
